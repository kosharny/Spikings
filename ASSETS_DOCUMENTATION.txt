NOTES FOR DEVELOPERS
---------------------

1. All images should be provided in @1x, @2x, and @3x resolutions
2. Recommended format: PNG or JPEG
3. Color profile: sRGB
4. Images should maintain aspect ratio of approximately 16:9 for covers
5. Background images should be at least 1170x2532 pixels for iPhone compatibility
6. Consider using placeholder gradients during development
7. Final assets should be optimized for app size

PLACEHOLDER STRATEGY
--------------------

During development, the app uses SF Symbols and gradient backgrounds as placeholders.
The UI is designed to work with or without actual image assets.
Replace placeholders with actual images before production release.
